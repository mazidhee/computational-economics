This folder contains standalone scripts and notebooks for calculus-related experiments: numerical differentiation, optimization examples, numerical integration, and visualization.

Usage
- Open notebooks with JupyterLab / Jupyter Notebook
- Run small scripts directly:

```bash
cd calculus
python numerical_derivative.py
```

Dependencies
- Python 3.8+
- numpy, scipy, matplotlib (optional)

Notes
- These are learning-oriented artifacts and do not depend on the main pipeline.
